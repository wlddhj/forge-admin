# 项目截图

此目录存放 README.md 中引用的项目截图。

## 所需截图

请准备以下截图文件：

| 文件名 | 说明 | 建议尺寸 |
|--------|------|----------|
| `login.png` | 登录页面 | 1920x1080 |
| `dashboard.png` | 仪表盘/首页 | 1920x1080 |
| `user.png` | 用户管理页面 | 1920x1080 |
| `role.png` | 角色管理页面 | 1920x1080 |
| `menu.png` | 菜单管理页面 | 1920x1080 |

## 截图建议

1. 使用浏览器全屏模式或 1920x1080 分辨率
2. 隐藏敏感信息（真实用户名、密码等）
3. 使用示例数据而非真实业务数据
4. 保持截图风格统一

## 截图方法

### macOS
- `Cmd + Shift + 4` 选择区域截图
- `Cmd + Shift + 3` 全屏截图

### Chrome 开发者工具
1. 打开开发者工具 (F12)
2. 按 `Cmd/Ctrl + Shift + P`
3. 输入 `screenshot`
4. 选择 `Capture full size screenshot`
